# 🧮 Python Calculator

A simple and beginner-friendly calculator built using Python.  
This project performs basic arithmetic operations with user input through the command line.

## ✨ Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero errors

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Make sure Python is installed
2. Download or clone this repository
3. Run the file using:
   python calculator.py

## 📌 About
This project was created as a beginner practice to understand basic Python concepts like:
- User input
- Conditional statements
- Basic operations
